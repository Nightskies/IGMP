IGMP CLIENT

Использование:
make - собрать все необходимые файлы
Для запуска нужно перейти в папку bin и ввести:
sudo ./client [OPTINS] group1 group2 [...] interface

Чтобы посмотреть описание доступных комманд:
./client -h

Чтобы использовать дополнительную отладочную информацию.
./client -d group1 group2 [...] interface
